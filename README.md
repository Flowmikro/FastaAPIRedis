# FastaAPIRedis
____
## Установка  
Установить репозиторий
```commandline
git clone https://github.com/Flowmikro/FastaAPIRedis.git
```
Перейти в директорию
```commandline
cd FastaAPIRedis
```
Запустить докер 
```commandline
docker-compose up -d
```
Перейти по адресу
```commandline
http://localhost:8000/docs
```
